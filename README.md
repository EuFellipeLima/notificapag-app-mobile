# 📱 NotificaPag - Mobile App

Aplicativo front-end desenvolvido em React Native (Expo) para consulta e notificação de parcelas e vencimentos. Criado focado na experiência do usuário B2B2C, permitindo acesso rápido a dados de faturamento sem a necessidade de senhas complexas, utilizando autenticação baseada em dados cadastrais (KBA).

## 🚀 Tecnologias Utilizadas
* React Native
* Expo Go
* JavaScript/TypeScript
